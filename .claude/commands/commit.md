---
description: Atomic Commit原則に従いセマンティックコミットを作成
---

## 準備

以下のドキュメントを読み込んでください：
- `~/.claude/docs/git/commit-format.md`
- `~/.claude/docs/git/atomic-commit.md`

## 実行

1. git status と git diff で変更内容を確認
2. Semantic Commit形式でコミットメッセージを生成
3. git commit でコミット実行

ドキュメントに従い、Atomic Commitの原則を守ってコミットを作成してください。
